NPL

Normative Programming Language (NPL) is a language to program the norms of multi-agent systems. Some papers describing the language are available in the folder publications.

The folder examples contains Java programs to illustrates its use.

This language is used to implement the organisational platform for Moise, called ORA4MAS, that is integrated into the JaCaMo project.

Installation

Some built releases are available at GitHub and the source code can be used with the following commands.

git clone https://github.com/jomifred/npl.git
cd npl
gradle build
gradle eclipse

To run the examples:

gradle example0
gradle example1
...
